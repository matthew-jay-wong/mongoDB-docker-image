# MongoDB Docker Image

## Accounts

| Type | Username | Password     |
| ---- | -------- | ------------ |
| root | root     | rootpassword |
| user | username | password     |

## Ports

Connect through localhost to the following ports: `27017-27019`

## Commands

Bring up docker instance:

```bash
npm run start
```

Bring down docker instance:

```bash
npm run stop
```
